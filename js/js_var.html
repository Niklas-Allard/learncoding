Variabeln



1. Schritt

Zuerst muss die Variabel deklariert werden, dazu nutzt man let 

                let x;

2. Schritt

Dann wird die Variabel vergeben: 

                x = 10;


Variabeln in Strings einfügen

Man kann mit einem Dollar-Zeichen($) dahinter einer Klammer und in der Klammer eine Variabel, die Variabel in einem String ausgeben. 


                let age = 25;

                console.log("You are $(age) old");


Alle Datentypen

  1. Numbers (Zahlen auch float)
  2. String
  3. Boolean


Datentyp einer Variabel ausgeben

Um den Datentyp einer Variabel zu erhaloten muss man bloß die "typeof var" Methode benutzten, in dem Fall steht var für egal welche Variabel



Datentyp umwandeln

x = Number(x)
y = String(y)
z = Boolean(z)
