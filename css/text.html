Text



color:
  • Farbnamen: purple, lightgreen, red
  • RGB-Werte: rgb(120,180,120)
  • Hexadezimal: #2596BE (das Hashtag muss da sein)


text-transform:
  • none (keine Texttransformation)
  • capitalize (der Erste Buchstabe jedes Wortes wird groß geschrieben)
  • uppercase (Alle Buchstaben werden groß geschrieben)
  • lowercase (Alle Buchstaben werden klein geschrieben)


text-indent:
  • Länge oder Prozent
Ist eine Einrückung eines Textes wie TAB

text-decoration:

Wie bei Padding ist "text-decoration" der Überbegriff/Gruppe von "text-decoration-line", "text-decoration-style-style" und "text-decoration-color". text-decoration-line:

Erstellt eine Linie über, unter oder durch den Text:
  • none
  • underline
  • overline
  • line-through


text-decoration-style:

Aussehen der Linie
  • solid (normale Linie)
  • dotted (gepunktete Linie)
  • dashed (gestrichelte Linie)
  • wavy (gewellte Linie)


text-decoration-color:
  • Farbe


Beispiel:


                a {
                    text-decoration: solid dotted red
                }


text-shadow:
  • color
  • offset-x offset-y (Damit kann man einstellen, wo der Schatten sein soll)
  • blur-radius (macht den Schatten verschwommen)


letter-spacing:

Damit kann man einstellen wie viel Platz zwischen Buchtstaben ist
  • normal
  • px
  • prozent


word-spacing:

Wie bei Letter-spacing nur mit Wörtern

line-height:
  • normal
  • Zahl
  • px
  • Prozent


white-space:

Damit kann man Leerzeichen modifizieren
  • normal
  • nowrap (kein automatischer Zeilenbruch; <br> wird beachtet)
  • pre (Leerzeichen bleiben unverändert)
  • dircetion
    • ltr (der Text geht von links nach rechts)
    • rtl (der Text geht von recht nach links)
