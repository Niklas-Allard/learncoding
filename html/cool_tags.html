Interessante Elemente




Akkronyme anzeigen

<abbr></abbr> Man kann mit diesem Element die Bedeutung von Akkronymen anzeigen

Beispiel:


            <abbr title="Hypertext Markup Language">HTML</abbr>


Output:

HTML (hover)

Vorschläge bei input-Inputs

Man kann mit diesen folgenden Elementen in einem Input Text-Feld Vorschläge anzeigen. Das wird an folgendem Beispiel erklärt. 


            <input type="input" list="colors">
            <datalist id="colors">
                <option value="red"></option>
                <option value="orange"></option>
                <option value="blue"></option>
                <option value="dark green"></option>
            </datalist>




Genau hier drüber kann man den Output sehen vom Code. um die Dataliist mit dem Input-Feld zu verbinden, muss das Attribute list genau den selben Wert wie id haben. Einfacher Weg eine Box zu erstellen

Mit den beiden Elementen <fieldset> und <legend> ist es einfach schöne und praktische Boxen zu erstellen.


            <fieldset>
                <legend>Echt schön</legend>
                Berlin, Paris
            </fieldset>


Output:

Echt schön Berlin, Paris


Klar sieht es gerade nicht richtig schön aus aber es man kann das ganz einfach mit ein bisschen CSS beheben.
