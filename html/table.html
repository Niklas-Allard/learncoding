Tabellen



<table> </table> Anfang und Ende von einer Tabelle

<tr> </tr> Table row Zeigt das eine neue Tabellenreihe beginnt oder endet
<th> </th> Table Head oben auf der Tabelle
<td> </td> Table data da zwischen kommt der Text

Beispiel:


            <table> 
                <tr> 
                    <th> Ich  
                    <th> Du  
                </tr> 
                <tr> 
                    <td> 0  
                    <td> 1  
                </tr> 
            </table>

Output:

Ich Du
0 1
